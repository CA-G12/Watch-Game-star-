# animovie

Our application allows the user to download movies and anime

## Links

- Live Link [Here](https://ca-g12.github.io/Watch-Game-star-/)

## Problem

We were using a gaming API, and after agreeing and starting to implement the idea, we discovered that it had broken

## Solution

We used the alternative idea, and used the API of Anime

## User Stories

-I want to see the lists of movies and anime
-I want to download movies and anime
-I want to put my favorite movies and anime as favourites

## How To clone Our App:

- Open gitBash wherever you want to save the project on your device.
- Clone the repository to your device (`git clone repo link`).
- Enter repo `cd nameOfRepo`
- Type -> `code .`
- Navigate to the live server from the HTML file.

## How Test Our App

- Open gitBash where project path is.
- Type -> `node install`
- then Type -> `npm test`
